

# Responsive Navbar Project

## Purpose
For students learning HTML, CSS, and frameworks. This project is an open-source resource for learning and experimentation—feel free to clone and use it.

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- Vanilla JavaScript

## Structure
- `HTML/` — Contains `index.html` (main markup)
- `CSS/` — Contains `style.css` (all styles, responsive and theme logic)
- `JS/` — Contains `script.js` (all interactive logic)

## Features
- Fixed, responsive navigation bar (hamburger menu for mobile, horizontal for desktop)
- Animated hamburger icon
- Smooth dark/light theme toggle using CSS variables
- SVG-based favicon ("Ac" logo branding)

## Git
- Standard `.gitignore` for web projects is included (ignores node_modules, .DS_Store, build outputs, etc.)
